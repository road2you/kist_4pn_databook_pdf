1  전체 아이템 구조

1.1 미래유망 융합기술 아이템의 영역 유형화

  • 최종 371개 아이템은 CWTS Leiden Ranking (2023)에서 발표한 4,215개 클러스터를 대상으로 다양한 관점의 계량지표 분석을 수행하고 결과를 종합하여 선별한 것임.
    • 클러스터별 지표 값은 대시보드로 구축하여 탐색과 비교를 지원
    • 선별과정의 구체적인 내용은 부록 1에서 정리


  • 371개 아이템(클러스터)들을 유형화하기 위해 클러스터 간 인용(cross citation network) 데이터를 기반1으로 2단계 군집화를 수행하여 문헌들의 집합인 클러스터(micro) 수준을 포괄하는 상위의 43개 세부영역(meso)과 3개 대영역(macro) 분류기준을 도출하였음.
    • 세부영역과 대영역의 명칭은 LLM 기반 리포트 정보를 바탕으로 LLM 피드백과 결과의 세부 조정과정을 거쳐 부여
    • 클러스터 간 인용-피인용 정보에서 클러스터간 코사인 유사도를 생성하는 방식은 Zhang, Rousseau, and Glänzel (2016) 을 준용
    • 클러스터링 기법은 Leiden Algorithm(Traag, Waltman, and Eck 2019)을 적용

1 2017-2021년 5개년 간의 누적 인용 데이터를 적용


  • 3대 대영역별로 43개 세부영역과 최종 선별 371개 클러스터들이 어떻게 분포하는지는 표 1.1 에서 요약하였음.
    • 편의를 위해 대영역에 대한 약칭을 1번은 ‘지속가능 환경’, 2번을 ‘스마트인프라’, 3번을 ‘디지털 융합’으로 함.
    • 371개 아이템의 LLM 기반 리포트는 대영역-세부영역 기준으로 수록
      • 부록 2에서 리포트 내용 생성 과정 정리

1.2 미래유망 융합기술 매핑

  • 세부영역(meso) 분류 간의 유사도 기반 매핑을 VOSviewer를 이용하여 수행하고 대영역(macro) 구분을 그림 1.1 과 같이 시각화함.
    • 앞서 설정한 약칭 기준으로 1번 ‘지속가능 환경’은 우측의 초록색 영역, 2번 ‘스마트 인프라’는 가운데 파란색 영역, 3번 ‘디지털 융합’은 왼쪽 빨간색 영역
    • 탭을 이용해 온라인 버전에 연결할 수 있음.
    • 화면에서 세부영역 간의 링크는 상위 200개만 보이도록 설정하였으며, 노드의 크기는 해당 세부영역에 포함된 2020-2023년 4년 간의 누적 문헌수를 나타냄.
  • 매핑 (캡처)
  • 매핑 (온라인)

1.3 아이템 분류 총괄 연계표

  • 대영역(macro), 세부영역(meso), 최종 미래유망 융합기술 아이템(micro)의 연계표는 표 1.2 에서 확인할 수 있음.
    • label_short_new는 리포트 생성 단계에서 보정한 클러스터(아이템) 이름이며, label_long_kr은 기존 대시보드에 수록된 클러스터 이름을 가리킴.
